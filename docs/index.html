Index

A simple 2D collision detection engine.

Install

$> npm install konjo

Current features

  • Base math lib (vector2/axes)
  • Base box object (AABB/OBB/Sphere)
  • Lightweight

How to build

$> npm install

Then, to build the source, run:

$> npm run build
